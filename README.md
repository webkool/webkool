# Webkool
***
open-source platform to develop client and/or server web application.

## Install

    npm install -g webkool
    
## Usage

    wkc [options] file.wk

  options:
  
-   **--server**	compile server based application
-   **--client**	compile client based application
-   **-i**		include directory
-   **-o**		output basename
-   **--hint**		jshint config file
